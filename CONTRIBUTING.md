# Panduan Kontribusi

Terima kasih telah tertarik untuk berkontribusi pada Antivoid E-Commerce! Kami sangat menghargai bantuan Anda dalam membuat proyek ini menjadi lebih baik.

## Cara Berkontribusi

1.  **Fork** repositori ini.
2.  Buat **Branch** baru untuk fitur atau perbaikan Anda (`git checkout -b fitur/fitur-keren-saya`).
3.  Lakukan **Commit** perubahan Anda dengan pesan yang jelas (`git commit -m 'Menambahkan fitur XYZ'`).
4.  **Push** ke branch Anda (`git push origin fitur/fitur-keren-saya`).
5.  Buka **Pull Request** ke repositori asli.

## Aturan Penulisan Kode

- Pastikan kode Anda mengikuti gaya penulisan Ruby yang konsisten.
- Selalu uji perubahan Anda dengan menjalankan server lokal sebelum mengirimkan Pull Request.
- Untuk perubahan UI, pastikan tetap mengikuti tema **Glassmorphism** dan desain yang sudah ada menggunakan Tailwind CSS.

## Melaporkan Masalah (Issues)

Jika Anda menemukan bug atau memiliki saran fitur, silakan buka Issue baru dengan menyertakan:
- Deskripsi singkat masalah/saran.
- Langkah-langkah untuk mereproduksi bug.
- Hasil yang diharapkan vs hasil yang sebenarnya.

Keamanan Anda dan kenyamanan pengguna adalah prioritas kami. Mari bangun ekosistem open-source yang hebat!
